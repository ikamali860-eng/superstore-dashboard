# superstore- Streamlit dashboard 
## Project Overview
This Project is an interactive Sales Dashboard built using Python to analyse retail data from the SuperStore dataset.
The goal is to explore sales performance across regions, categories, and time, and extract usefull business insights.
## Objectives 
- Track overall sales performance using key metrics (KPIs)
- Identify top-performing products and categories
- Analyze sales distribution across regions
- Detect trends and seasonality in sales over time
- Apply the Pareto principle (80/20 rule) to understand revenue concentration
## Features 
## KPI (Key Metrics) 
- Total Sales
- Total Orders
- Average Sales
## Visualizations
- Sales by Category
- Sales by Region
- Sales Over Time
- Top 10 Products
## Interactive Filters
- Region Filter
- Date Range Filter
# Advanced Analytics 
- Customer Segment Analysis
- Sales Heatmap (Month vs Category)
- Monthly Sales Trend
- Pareto Analysis (80/20 rule)
## Tools
- Python
- Pandas
- Plotly
- Streamlit
## Dataset
The dataset used is the SuperStore Sales Dataset, which contains:
- Orders
- Products
- Customers
- Sales data
## Key Business Insights
- A small percentage of products contributes to the majority of total sales(Pareto principle)
- Sales performance varies significantly across regions, indicating potential market opportunities
- Certain product categories consistently outperform others
- Sales trends show seasonal patterns that can guide inventory and marketing startegies
- Customer segments contribute differently to revenue, highlighting target market opportunities
